Commissioner

History of the font

Commissioner is a humanist sans-serif font, that was created by Kostas Bartsokas it is a variable font that contains four axes flare, slant, weight and volume. Each one of these axes gives designers the opportunity to add their own voices to the typeface without changing it entirely. The most interesting of the four axes are flare and volume. Flare allows designers to adjust terminal shapes and add “flare”, whereas volume modifies the Commissioners serifs to create either angular or softer corners.
